A Python-based Language Translation Tool with a user-friendly Tkinter GUI that translates text between multiple languages using the Deep Translator API.
# Language Translation Tool

A Python GUI application that translates text between multiple languages.

## Features
- User-friendly interface
- Multiple language support
- Real-time translation
- Built using Tkinter and Deep Translator

## Technologies Used
- Python
- Tkinter
- deep-translator

## How to Run

pip install deep-translator

python language_translator.py
