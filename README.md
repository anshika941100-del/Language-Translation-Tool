# 🌍 Language Translation Tool

A simple and user-friendly Language Translation Tool built using Python, Tkinter, and Deep Translator. This application enables users to translate text between multiple languages through an intuitive graphical user interface.

## ✨ Features

- 🌐 Translate text between multiple languages
- 🖥️ User-friendly Tkinter GUI
- ⚡ Fast and efficient translation
- 🎯 Simple and beginner-friendly design
- 📝 Easy text input and output
- 💻 Lightweight and easy to use

## 🛠️ Technologies Used

- Python 3.14
- Tkinter
- Deep Translator

## 📂 Project Structure

```text
Language-Translation-Tool/
│── language_translator.py
│── requirements.txt
│── README.md
```

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/anshika941100-del/Language-Translation-Tool.git
```

2. Go to the project directory:

```bash
cd Language-Translation-Tool
```

3. Install the required packages:

```bash
pip install -r requirements.txt
```

## ▶️ Run the Application

```bash
python language_translator.py
```

## 💡 How to Use

1. Enter the text you want to translate.
2. Select the source language.
3. Select the target language.
4. Click the **Translate** button.
5. View the translated result in the output box.
